# Numerical Methods in Fortran

This repository contains various numerical methods and computational mathematics algorithms implemented in **Fortran 90/95**.

## 📂 Available Methods

* **Root-Finding:** Bisection, False Position, Fixed-Point, Newton-Raphson, and Secant Methods.
* **Linear Systems:** Gaussian Elimination, Gauss-Seidel, and Jacobi Iteration.
* **Approximation:** Chebyshev Polynomials.

## 🚀 How to Run

Open your terminal in this directory and run:

```bash
# Compile
gfortran filename.f90 -o program

# Run
./program
